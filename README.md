# GitHub Automation Toolkit

A shell script project to automate GitHub tasks.
Made by: Abhay Thakare (Abhay-aiml)

## Features
- Auto commit and push to GitHub
- Backup all GitHub repos locally
- Show commit stats for all repos

## Scripts
- main.sh - Main menu
- auto_push.sh - Auto push to GitHub
- repo_backup.sh - Backup all repos
- commit_stats.sh - Show commit stats

## How to Run
bash main.sh

## Skills Used
- Shell Scripting
- GitHub API
- Git Automation
- Linux/Ubuntu
