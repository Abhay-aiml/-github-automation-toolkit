# 🚀 GitHub Automation Toolkit

A powerful shell script project to automate GitHub tasks built on Ubuntu Linux.

## 📌 Features
- ✅ Auto commit and push to GitHub
- ✅ Backup all GitHub repos locally
- ✅ Show commit stats for all repos
- ✅ Beautiful color coded terminal output
- ✅ GitHub API integration

## 🛠️ Tech Stack
- Shell Scripting (Bash)
- GitHub REST API
- Git
- Linux/Ubuntu

## 📁 Project Structure
main.sh          - Main menu interface
auto_push.sh     - Auto commit and push
repo_backup.sh   - Backup all repos
commit_stats.sh  - Show commit stats
variables.sh     - Variables lesson
conditions.sh    - If/Else lesson
loops.sh         - Loops lesson
functions.sh     - Functions lesson

## ▶️ How to Run
git clone https://github.com/Abhay-aiml/github-automation-toolkit.git
cd github-automation-toolkit
chmod +x main.sh
./main.sh
